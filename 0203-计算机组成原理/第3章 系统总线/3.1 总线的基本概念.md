
# 总线优点

- 复杂部件间的控制和数据交换

- 统一各部件的接口

- 容易扩展新的部件

# 总线定义

总线是连接各个部件的信息传输线，是各个部件共享的传输介质

# 信息传输方式

- 串行：数据在一条线上连续传输

- 并行：数据分散在多条线上同时传输，线路之间容易产生干扰
